== Description ==

This plugin disables the new emoji functionality in WordPress 4.2. It also removes some additional meta tags.

== Installation ==

After you've downloaded and extracted the files:

1. Upload the complete 'remove-emojis-meta' folder to the '/wp-content/plugins/' directory OR install via the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
